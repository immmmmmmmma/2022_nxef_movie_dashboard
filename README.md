# movie

영화 대시보드 만들기

<br/>

# 로컬 실행 명령어

```
# 패키지 설치
$ yarn

# 로컬 실행
$ yarn start

```

<br/>

# 기능 목록

1. 로그인 페이지

   1. 이메일 형식 아이디

   2. 영문 숫자 특수문자 3종류 이상을 조합하여 최소 8자리 

   3. 탭 내에서 로그인 유지

<br/>

2. MovieList 페이지

   1. 목록 리스트 구현<br/>
      (title,img,vote_average,vote_count,release_data,overview)

   2. 목록에 항목 추가

   3. 목록 항목 수정

   4. 목록 항목 삭제

<br/>

3. 검색 페이지

   1. 검색 필터 구현(movie, people, tv)

   2. 검색 시 각 데이터에 맞는 필드 구현
